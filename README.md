# AI Resume Builder

AI Resume Builder is a production-ready Streamlit web application that transforms raw career data into ATS-optimized, professional DOCX resumes. Paste a job description to receive real-time ATS scoring (TF-IDF cosine similarity), AI-powered skill gap analysis, and keyword matching powered by Groq's Llama-3.3-70b. Automatically enhance project descriptions with quantifiable achievements and generate smart skill recommendations. Preview live HTML resumes with professional typography. Export polished DOCX files with precise formatting that passes ATS parsers. Graceful fallbacks ensure full functionality without API keys. Built with Streamlit, python-docx, scikit-learn. Deploy instantly: `pip install -r requirements.txt && streamlit run app.py`. Open-source, hackathon-proven.

## Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```

- ✅ ATS Score (TF-IDF + keyword matching)
- ✅ AI Resume Coaching (Groq Llama-3.3)
